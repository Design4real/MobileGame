This package contains a detailed container crane.
The crane is segmented into 4 parts.
Just drag the prefab into your scene.

You may also want to have a look at the example scene.

Textures-sizes 2048x2048
Textures are created using Ambient-Occlusion.
Total Polygoncount 40.000

The original FBX-file is also included in the package.
